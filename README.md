# Ayush Portfolio V2 - Music Edition

A modern, music-themed portfolio website built with React, TypeScript, and Tailwind CSS.

## Features

-  Music player-inspired UI with vinyl aesthetics
-  Bold typography and neon color palette
-  Smooth animations with Framer Motion
-  Fully responsive design
-  Accessible and keyboard-friendly
-  Album cover-inspired project cards
-  Audio visualizer effects
-  Instrument/equipment themed skills section

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Customization

- Update your information in the component files
- Modify colors in `tailwind.config.js`
- Add your projects in `src/components/Projects.tsx`
- Update skills in `src/components/Skills.tsx`
- Add experience in `src/components/Experience.tsx`
